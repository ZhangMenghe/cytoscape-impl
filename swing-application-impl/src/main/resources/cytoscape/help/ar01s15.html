Apps and the App Manager

The App Manager

The App Manager, first introduced in Cytoscape 2.5, allows users to quickly and conveniently add extra features to Cytoscape directly from within Cytoscape, eliminating the need for manual searches through different webpages to install and update apps.

For app developer, to enable automatic download of your app to Cytoscape users, your app should be in compliance with Cytoscape 2.5, and the app jar/zip files should be uploaded to the Cytoscape app web site at http://cytoscape.org/apps25/index.php.

Note: If you do not have Internet access enabled, you will not see the list of available apps or be able to automatically update existing ones; however, you will still be able to view and delete previously installed apps.

Get New Apps

To install new features, go to the App Manager at Apps → Manage Apps. On the left side of the window that pops up, you will see app folders labeled Currently Installed and Available for Install. Double-clicking on these will show sub-folders, and then the apps themselves. To find out more about a specific app, click on its name to display some basic information on the right-hand side of the window.

The Currently Installed folder contains a number of default apps that are fully integrated in every copy of Cytoscape, as well as any additional installed apps. In contrast, the Available for Install folder displays apps that may be installed. To install and use these apps, click on the file name and then click on the Install button at the bottom of the window. A license agreement may appear, in which case you must accept it in order to download the app. You will then see a progress bar as the app is automatically downloaded and installed into your current version of Cytoscape. The progress bar will disappear when the download is complete. Other apps can then be downloaded, or the manager can be closed by clicking the Close button.

Once a new app is added, it can immediately be used. Closing and restarting Cytoscape is not required. For example, installing the MCODE app will automatically result in new menu options such as Apps → MCODE → Start MCODE.

Troubleshooting

  • If an installation error appears, automatic installation of the app may not be supported. To manually install the app, go to the Cytoscape apps page (http://cytoscape.org/apps25/index.php), scroll down to find the app, click on the appropriate link to download the file, and then save it in the Cytoscape/apps folder on your hard drive.

  • If the new app does not automatically work, you may need to restart Cytoscape.
  • If the app does not appear in the Currently Installed folder of the App Manager, then Cytoscape was unable to load the app. Your command line will display the error message generated.

Delete Existing Apps

To delete an existing app, go to the App Manager at Apps → Manage Apps, click on the appropriate app once to select it, and then click the Delete button at the bottom of the manager. It is best to avoid deleting core apps, as this may cause Cytoscape or other apps to malfunction. A app will be completely deleted after you have restarted Cytoscape.

Update Existing Apps

Going to Apps → Update Apps will result in Cytoscape automatically accessing the Internet to look for updated versions of the apps you have installed. If newer versions are available, Cytoscape will then display the available updates in the App Manager; otherwise, it will inform you that there are no updatable apps. When multiple updates are detected, you will have the option to download some or all of the updates. Some updates may require re-accepting the license before the download will begin. All updates require Cytoscape to be restarted in order for the update to be fully installed. After clicking the Update All or Update Selected buttons, a progress bar will appear showing the status of the download; it will disappear once the update is complete, and the App Manager can then be closed by clicking the Close button.

Note: The Update All button will update all listed apps to the newest available version. If you prefer a different version, select the name and click Update Selected.

Download Apps from a Custom Site

The App Manager automatically connects to the Cytoscape website and the sites of registered Cytoscape apps to check for and download any available apps and updates. To change this setting, open the App Manager (by going to Apps → Manage Apps) and click the Change Download Site button. You will be asked to choose a site from a preset list of bookmarks. Select one and click Ok. Alternatively, click the Add Site button to open the Bookmark Manager and modify the list of bookmarks. In the Bookmark Manager, choose the apps category from the dropdown list, and then add/edit/delete URLs using the appropriate buttons on the right-hand side of the window. After the correct site is selected, set it as the download site by clicking OK. The App Manager will refresh using that site.

Note: Custom download sites added from the App Manager will not be saved when you exit Cytoscape.
